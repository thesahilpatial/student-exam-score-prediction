# 📊 Student Exam Score Prediction

This project predicts student exam scores using Machine Learning techniques.

## 🚀 Project Workflow
1. Data Exploration (EDA)
2. Data Cleaning & Preprocessing
3. Correlation Analysis
4. Feature Engineering
5. Train-Test Split
6. Model Training
7. Model Evaluation

## 🤖 Machine Learning Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## 📈 Model Performance

| Model | R² Score | MAE |
|------|------|------|
| Linear Regression | 0.777 | 7.09 |
| Random Forest | 0.767 | 7.23 |
| Gradient Boosting | **0.780** | **7.06** |

Gradient Boosting performed the best.

## 📊 Features Used

- Study Hours
- Class Attendance
- Sleep Hours
- Sleep Quality
- Study Method
- Facility Rating
- Internet Access
- Course
- Gender

## 🛠 Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🎯 Outcome

The trained model predicts exam scores for unseen students and generates a submission file containing student IDs and predicted scores.
